Welcome to the ABAdmin Free AngularJs Bootstrap 3 template

AB Admin is the most extensive WebApp template for admin dashboards and control panels. It is a responsive HTML template that is based on the CSS framework Bootstrap 3+ and the javascript framework AngularJs 1.7.2. It utilizes all of the Bootstrap components in its design and re-styles many commonly used plugins to create a consistent design that can be used as a user interface for backend applications. All the CSS styles used are defined in LESS files for easy use. AB Admin is based on a modular design, which allows it to be easily customized and built upon.

The template is the only one where the clients / users will be offered to decide which demo apps that should be included into the App Views.

AngularJS version

Angular version is built with the latest technology contains well organized and coded controllers and directives written specifically to support the angular framework (AngularJS 1.6.9). This version is writen with Opinionated AngularJS styleguide for teams. And it is provided with full Grunt/Bower environment.
